## ⭐주요 기능
✅ 공고 관련
- 공고 등록
- 공고 조회(전체, 개별)
- 공고 수정
- 공고 삭제(기업회원만)
  
✅ 이력서 관련
- 이력서 작성
- 이력서 조회(전체, 개별)
- 이력서 수정
- 이력서 삭제(개인회원만)

✅ 지원 관련
- 지원하기
- 이력서 확인
- 모집 종료
- 채용 인원 자동 선발
- 지원 결과 확인(PASS, FAIL)
